# Stonker

## Author: Ian Lingo

### Project Description:
Stonker is a stock analyzer web application that allows users to gain insight on specific stocks of their choosing. Stonker will allow users to bookmark and save the stocks of their choosing locally and will later allow for user login capabilities. The web app will supply a data visual of the stock and its market trends to allow the user to see the data clearly and interact with the graphical user interface. Stonker's UI/UX will be seamless, simple, and elegant in design. The web app should be easily customized to the users choice and allow for more additional features in later iteractions/versions...

## Project Outline: 

#### Interface Plan: 
The front end will be entirely in Python with Flask routing and web interface. The GUI will be interactive and fully customizable to the users will. 

#### Data Collection and Storage Plan: 
Pandas wil be used to clean and organize the data from a stock market API (TBD). CSV and/or JSON will allow for local storage for the user (save stock insights and page).

#### Data Analysis and Visualization Plan: 
NumPy and Plotly/Matplotlib will be used for data analysis and data visualization. The stock market data will come from a stock market API which is then filtered to the users request (buttons and navbars user interacts with).
